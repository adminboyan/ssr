# 使用教程

## debian or ubuntu
```shell
apt-get install curl -y
bash <(curl -sSL "https://raw.githubusercontent.com/iboyan/ssr/main/node.sh")
```

## centos7

```shell
yum install curl -y
bash <(curl -sSL "https://raw.githubusercontent.com/iboyan/ssr/main/node.sh")
```
